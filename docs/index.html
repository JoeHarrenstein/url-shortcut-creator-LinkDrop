LinkDrop

Create URL shortcuts with one click.

A free Windows utility for creating .url shortcut files with automatic favicon fetching. Perfect for shared network drives and organized bookmarks.

Download for Windows View on GitHub
// FEATURES

Everything You Need

Single & Batch Modes

Create one shortcut at a time or process multiple URLs in batch mode for maximum efficiency.

Automatic Favicons

Fetches high-quality 256px favicons automatically using Google's favicon service for crisp icons.

Clipboard Auto-Detect

Automatically detects URLs in your clipboard when you open the app for instant shortcut creation.

Right-Click Integration

Optional Windows context menu integration lets you create shortcuts directly from any folder.

Duplicate Warning

Warns you before overwriting existing shortcuts, preventing accidental data loss.

Network Path Support

Full UNC path support for saving shortcuts directly to shared network drives and mapped locations.
